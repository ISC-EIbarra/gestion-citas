## Gestión de citas

> Seguimiento de pacientes de una veterinaria, implementa LocalStorage para la persistencia de los datos al recargar el navegador.

<br>

**Operaciones**

```
- Agregar un paciente.
- Editar la información registrada del paciente.
- Eliminar al paciente.
```

**Tecnologías**

```
- React JS
- Tailwindcss
- Eslint + Airbnb
```
